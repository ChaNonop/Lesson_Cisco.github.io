# การปรับปรุงระบบนำทาง สไตล์สี และการจัดรูปแบบ Reference

ในการปรับปรุงรอบนี้ ได้ดำเนินการแก้ไขส่วนติดต่อผู้ใช้ (UI/UX) และระบบนำทางหลักตามความต้องการดังนี้:

1. **ปรับโครงสร้าง Sidebar**:
   - เพิ่มขนาดความกว้างจาก `w-72` เป็น `w-80` เพื่อป้องกันข้อความยาว ๆ หักบรรทัดและให้อ่านได้สะดวกยิ่งขึ้น
   - นำป้ายสถานะ "พร้อมเรียน" ของบทเรียนที่เปิดแล้วออกตามสั่ง เพื่อลดความหนาแน่นของอินเตอร์เฟส
   - เปลี่ยนป้ายเฟสของบทเรียนที่ยังไม่มีหน้าเว็บ (บทที่ 7 - 9) เป็น **"กำลังดำเนินการ"** สีส้ม-ทอง เพื่อให้ผู้เรียนทราบล่วงหน้าชัดเจน
   - ปรับสีตัวอักษรของลิงก์ที่ยังไม่ได้คลิกในธีมสว่าง จากสีเทาอ่อน `text-slate-400` เป็นสีเทาเข้ม `text-slate-700` เพื่อให้เห็นชัดเจนและลดอาการปวดตา

2. **ระบบนำทางย่อยแบบแท็บหัวข้อ (Subtopic Dynamic Links)**:
   - เพิ่มข้อมูล `subtopics` (เช่น 1.1, 1.2, 5.1) ลงในชุดข้อมูลบทเรียนของไฟล์ `lessons.js`
   - พัฒนาโค้ดจาวาสคริปต์ใน `app.js` ให้ตรวจหาและสร้างลิงก์ย่อยภายใต้บทเรียนที่กำลังเปิดอยู่อัตโนมัติใน Sidebar
   - เมื่อคลิกลิงก์ย่อย หน้าเว็บหลักจะทำแอนิเมชันเลื่อนหน้าจอ (Smooth Scroll) ไปยังเนื้อหาหัวข้อนั้น ๆ ทันที

3. **ปรับโฉมใหม่ Quick Reference**:
   - แปลงโฉมหน้าไฟล์ **`reference/cisco-switch-quick-reference.html`** ให้มีธีมสีม่วง-น้ำเงินสไตล์เดียวกับหน้าหลักพอร์ทัล
   - เพิ่มแถบเมนูข้าง (Sidebar) แยกประเภทคำสั่งปฏิบัติการด่วน (การตั้งค่าพื้นฐาน, การเซฟ/ลบระบบ, Remote SSH, ขั้นตอนกู้รหัสผ่าน)
   - ปรับปรุงให้แสดงผลสวยงามและคมชัดทั้งในโหมดมืด (Dark Mode) และโหมดสว่าง (Light Mode) มีความสามารถในการกดคัดลอกคำสั่งได้ในตัว

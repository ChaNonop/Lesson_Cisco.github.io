# การเริ่มบทเรียนพื้นฐาน Cisco และแผนการเรียนรู้

ผู้เรียนได้ขอให้แนะนำเส้นทางการเรียนรู้สวิตช์และเครือข่าย Cisco รวมถึงคำสั่งพื้นฐานและการกู้คืนรหัสผ่าน (Password Recovery) จึงได้เริ่มต้นการสร้างบทเรียนแรกขึ้นมาเพื่อเป็นพื้นฐานในการเรียนรู้และทดสอบตนเองสำหรับหัวข้อสำคัญเหล่านี้ โดยปูรากฐานในการข้ามค่าตั้งค่าในการบูตเมื่อเกิดเหตุฉุกเฉิน

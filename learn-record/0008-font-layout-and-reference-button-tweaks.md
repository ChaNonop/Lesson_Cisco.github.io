# บันทึกความก้าวหน้า: การปรับปรุงฟอนต์ IBM Plex Sans Thai เลย์เอาต์หน้าเดี่ยว และปุ่ม Quick Reference

บันทึกรอบการพัฒนาเพื่อปรับเปลี่ยนการดีไซน์หน้าจอและการแสดงผลของระบบตามรายละเอียดดังนี้:

## 🛠️ รายละเอียดการปรับปรุง (Improvements)

### 1. แก้ไขปุ่มเปิด Quick Reference ในแถบข้าง (Sidebar) ของหน้าหลัก
* **ปัญหา:** คลาส `dark:bg-slate-850` ไม่ใช่ชื่อคลาสมาตรฐานของ Tailwind CSS ส่งผลให้ในโหมดมืด พื้นหลังปุ่มกลายเป็นโปร่งแสง/ดำ และกลืนไปกับพื้นหลังทำให้อักษรสีเทาของปุ่มมองไม่เห็น
* **วิธีแก้ไข:** ปรับเปลี่ยนคลาสใน [index.html](file:///D:/My_Doc_lesson/บทเรียน%20Cisco/index.html) ไปใช้งานคลาสมาตรฐาน:
  * เปลี่ยนเป็น `dark:bg-slate-800 dark:hover:bg-slate-700` สำหรับการควบคุมสีพื้นหลังในโหมดมืด
  * ปรับแต่งตัวอักษรเป็น `text-slate-800 dark:text-slate-200` พร้อมใส่กรอบ `border border-slate-300 dark:border-slate-700` เพิ่มมิติความลึกและการจัดวางที่สวยงามในทั้งสองธีม

### 2. ปรับขนาดของอักษรและโครงสร้างความกว้างของหน้าเดี่ยว (Lesson 1 Content Area Layout)
* **การดำเนินการ:** ปรับเพิ่มสเกลใน [lesson/0001-cisco-basics-and-password-reset.html](file:///D:/My_Doc_lesson/บทเรียน%20Cisco/lesson/0001-cisco-basics-and-password-reset.html)
  * ขยายขอบเขตความกว้างของเนื้อหาหลักจาก `max-w-3xl` เป็น **`max-w-6xl`** ทำให้แสดงเนื้อหาตาราง บล็อกข้อมูล และคำสั่งสวิตช์ได้กว้างขวาง สวยงาม สมสัดส่วนกับหน้าจอ Desktop
  * เพิ่มกฎ CSS เฉพาะในแท็ก `<style>` ของหน้านี้ เพื่อขยายขนาดฟอนต์โดยรวมขึ้นมาให้อ่านง่ายยิ่งขึ้น:
    * หัวข้อหลัก `h1` ปรับเป็นความกว้างเทียบเท่า `text-4xl/text-5xl` (~40px)
    * หัวข้อย่อยหลัก `h2` ปรับเป็นความกว้างเทียบเท่า `text-3xl` (~28px)
    * ข้อความบรรยายทั่วไป `p` ขยายเป็น `1.0625rem` (~17px)
    * ตารางและคำสั่งปฏิบัติการ ขยายขนาดการแสดงผลเพื่อการอ่านที่สบายตา

### 3. เปลี่ยนชุดฟอนต์เป็น IBM Plex Sans Thai
* **การดำเนินการ:** ดึงข้อมูลฟอนต์ภาษาไทยและอังกฤษยอดนิยม **`IBM Plex Sans Thai`** จาก Google Fonts เข้าสู่บทเรียนเดี่ยว พร้อมเซตตั้งเป็นฟอนต์หลักใน Tailwind Config
* **การสำรองข้อมูล (Font History Log):** เพื่อป้องกันการเปลี่ยนใจในอนาคต ได้ทำการบันทึกประวัติฟอนต์เก่าลงในโค้ดคอมเมนต์และ [NOTES.md](file:///D:/My_Doc_lesson/บทเรียน%20Cisco/NOTES.md):
  * **ฟอนต์เก่าก่อนหน้านี้:** `Outfit` (สำหรับภาษาอังกฤษ) และ `Sarabun` (สำหรับภาษาไทย)

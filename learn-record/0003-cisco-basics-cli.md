# การสร้างระบบจำลอง Cisco IOS CLI และบทเรียนการตั้งค่าเบื้องต้น

ในเฟสที่ 3 นี้ ได้พัฒนาและเปิดบทเรียนที่ 5 และ 6 ซึ่งเป็นบทปฏิบัติการจริงบนสวิตช์ Cisco:
1. บทเรียนที่ 5 ครอบคลุมการเชื่อมต่อสายคอนโซล และการนำทางในโหมดต่าง ๆ (User EXEC, Privileged EXEC, Global Configuration, Sub-modes)
2. บทเรียนที่ 6 ครอบคลุมคำสั่งตั้งค่าความปลอดภัยเบื้องต้น (Hostname, Enable Secret, Service Password Encryption, Banner MOTD) และความแตกต่างระหว่าง RAM (Running Config) กับ NVRAM (Startup Config)
3. มีการอัปเดตระบบการแสดงภาพโดยเพิ่ม **Cisco IOS CLI Terminal Simulator** ที่รองรับพฤติกรรมการพิมพ์คำสั่งของ Cisco จริง เช่น การย่อคำสั่ง (en, conf t, sh run, wr), การทำงานของระบบสลับโหมด, และการสืบค้นประวัติคำสั่งด้วยแป้นพิมพ์ลูกศรขึ้น/ลง

สิ่งนี้ช่วยสร้างความรู้พื้นฐานที่มั่นคงก่อนจะก้าวเข้าสู่การจัดการ Remote Management และ VLAN ในเฟสถัดไป

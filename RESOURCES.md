# แหล่งอ้างอิงและการเรียนรู้ Cisco (Cisco Configuration Resources)

## Knowledge (แหล่งเรียนรู้และคู่มือการใช้งาน)

- [Cisco Catalyst Configuration Guides](https://www.cisco.com)
  คู่มืออย่างเป็นทางการของ Cisco สำหรับการตั้งค่าอุปกรณ์สวิตช์แต่ละรุ่น ใช้เมื่อต้องการดูคำสั่งเฉพาะเจาะจงของฮาร์ดแวร์แต่ละซีรีส์
- [Cisco Catalyst 9000 Password Recovery Guide](https://www.cisco.com/c/en/us/support/docs/switches/catalyst-9000-series-switches/213795-password-recovery-procedure-for-catalys.html)
  คู่มือของ Cisco สำหรับการกู้คืนรหัสผ่านของอุปกรณ์ Catalyst 9000 Series (เช่น 9200, 9300, 9500) ซึ่งใช้ระบบ ROMMON แบบใหม่
- [Cisco Catalyst 2960/3560/3750 Password Recovery Guide](https://www.cisco.com/c/en/us/support/docs/switches/catalyst-2960-series-switches/12040-password-recovery-c2960.html)
  คู่มือกู้คืนรหัสผ่านดั้งเดิมสำหรับอุปกรณ์ซีรีส์ยอดนิยมรุ่นเดิม (Catalyst 2960, 3560, 3750) ที่ต้องกดปุ่ม Mode ค้างขณะบูต
- [NetworkLessons](https://networklessons.com)
  เว็บไซต์สอน Network ที่อธิบายเข้าใจง่าย มีรูปภาพประกอบชัดเจน เหมาะสำหรับการทำความเข้าใจแนวคิด เช่น VLANs, Trunking, VTP, STP
- [Cisco Packet Tracer (Networking Academy)](https://www.netacad.com/courses/packet-tracer)
  โปรแกรมจำลองเครือข่ายของ Cisco ที่ให้ดาวน์โหลดฟรี เหมาะสำหรับการทำแล็บทดลองทำตามคำสั่งโดยไม่ต้องมีอุปกรณ์จริง

## Wisdom (กลุ่มสังคมวิชาชีพและชุมชนแลกเปลี่ยนความรู้)

- [r/networking (Reddit)](https://reddit.com/r/networking)
  ชุมชนสำหรับวิศวกรเน็ตเวิร์กมืออาชีพ เหมาะสำหรับสอบถามข้อสงสัยในระดับการใช้งานจริง (Enterprise) การแก้ไขปัญหาที่ซับซ้อน และมาตรฐานการออกแบบระบบ
- [r/ccna (Reddit)](https://reddit.com/r/ccna)
  ชุมชนสำหรับผู้ที่กำลังศึกษาหรือสอบใบรับรอง CCNA มีผู้เชี่ยวชาญคอยตอบคำถามพื้นฐาน และแชร์แล็บจำลองสถานการณ์ต่างๆ
- [Cisco Learning Network](https://learningnetwork.cisco.com)
  คอมมูนิตี้อย่างเป็นทางการของ Cisco สำหรับการศึกษาหาความรู้และเตรียมสอบ มีเว็บบอร์ดถามตอบโดยวิศวกรของ Cisco และผู้เชี่ยวชาญทั่วโลก

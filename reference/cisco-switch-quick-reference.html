เอกสารสรุปคำสั่ง Cisco Switch Quick Reference

สรุปคำสั่ง คีย์ลัด และขั้นตอนด่วนสำหรับการตั้งค่าและกู้คืนรหัสผ่านสวิตช์ Cisco (สามารถพิมพ์หน้านี้เป็น PDF หรือกระดาษเพื่อใช้งานหน้างานได้)

1. ระดับสิทธิ์และโหมดการตั้งค่า (CLI Prompts)

  • User EXEC Mode Switch>
    • ใช้สำหรับดูสถานะทั่วไปเบื้องต้น เช่น การ ping หรือ traceroute
    • คำสั่งเข้า: เข้าถึงเมื่อเริ่มเชื่อมต่อพอร์ต Console
    • คำสั่งสลับไปโหมดถัดไป: enable
  • Privileged EXEC Mode Switch#
    • โหมดผู้ใช้งานระดับสูง สามารถรันคำสั่งตรวจสอบ (show) ล้างค่า ลบระบบ บันทึกคอนฟิก และดีบั๊กได้
    • คำสั่งเข้า: enable
    • คำสั่งสลับไปโหมดถัดไป: configure terminal
  • Global Configuration Mode Switch(config)#
    • โหมดตั้งค่าหลักของอุปกรณ์ ใช้แก้ชื่อ รหัสผ่าน สร้างบัญชี และสลับไปยังโหมดย่อย ๆ
    • คำสั่งเข้า: configure terminal
  • Interface Configuration Mode Switch(config-if)#
    • โหมดตั้งค่ารายพอร์ตหรือพอร์ตเสมือน (VLAN Interface)
    • คำสั่งเข้า: interface gigabitethernet 1/0/1 (หรือรุ่นอื่น ๆ เช่น interface fastethernet 0/1)

2. คำสั่งตรวจสอบระบบพื้นฐาน (Show Commands)

คำสั่งเหล่านี้ต้องรันในโหมด Privileged EXEC Switch# เท่านั้น:

  • show running-config - ดูคอนฟิกที่รันอยู่ใน RAM
  • show startup-config - ดูคอนฟิกที่บันทึกไว้ใน NVRAM (ค่าที่จะโหลดตอนบูตเครื่อง)
  • show ip interface brief - ตรวจดูสถานะ Up/Down และ IP Address ของพอร์ตทั้งหมดแบบสรุป
  • show interface status - ตรวจดูสถานะการเชื่อมต่อ ความเร็ว (Speed) และโหมดดูเพล็กซ์ (Duplex) ของพอร์ต
  • show vlan brief - แสดงรายการ VLAN และพอร์ตที่สังกัดอยู่
  • show mac address-table - ตรวจสอบตารางการจำค่า MAC Address ของสวิตช์
  • show version - แสดงข้อมูลฮาร์ดแวร์ เวอร์ชันซอฟต์แวร์ IOS และค่าคอนฟิกูเรชันรีจิสเตอร์

3. คำสั่งตั้งค่าเบื้องต้นจากโรงงาน (Basic Setup Workflow)

Switch# configure terminal
Switch(config)# hostname MySwitch
MySwitch(config)# enable secret MySecurePassword123
MySwitch(config)# service password-encryption
MySwitch(config)# banner motd # Unauthorized Access Prohibited! #

การตั้งค่า IP Address และ Default Gateway สำหรับเข้าจัดการ (SVI)

MySwitch(config)# interface vlan 1
MySwitch(config-if)# ip address 192.168.1.10 255.255.255.0
MySwitch(config-if)# no shutdown
MySwitch(config-if)# exit
MySwitch(config)# ip default-gateway 192.168.1.254

การเปิดใช้งาน SSH (Secure Remote Control)

MySwitch(config)# ip domain-name localnet.com
MySwitch(config)# crypto key generate rsa (เลือกคีย์ขนาด 2048 บิต)
MySwitch(config)# username admin privilege 15 secret MyAdminPassword
MySwitch(config)# line vty 0 15
MySwitch(config-line)# transport input ssh
MySwitch(config-line)# login local
MySwitch(config-line)# exit

การล้างการตั้งค่าโรงงาน (Factory Reset)

MySwitch# write erase
MySwitch# delete flash:vlan.dat (ตอบตกลงโดยกด Enter ยืนยัน)
MySwitch# reload

4. สรุปขั้นตอน Password Recovery ด่วน

แบบเดิม (ปุ่ม Mode)

  1. ถอดสายไฟสวิตช์
  2. กดปุ่ม Mode ค้างไว้แล้วเสียบสายไฟ
  3. ปล่อยปุ่ม Mode เมื่อ System LED ติดเขียวค้าง
  4. รันคำสั่ง flash_init
  5. รันคำสั่ง rename flash:config.text flash:config.text.old
  6. พิมพ์ boot
  7. พิมพ์ enable
  8. รัน rename flash:config.text.old flash:config.text
  9. รัน copy flash:config.text system:running-config
  10. เปลี่ยนรหัสผ่านใน configure terminal
  11. พิมพ์ copy running-config startup-config เพื่อบันทึก

แบบใหม่ (ROMMON Catalyst 9000)

  1. ต่อสาย Console เข้าสวิตช์
  2. บูตสวิตช์และกด Ctrl+C หรือกดปุ่ม Mode ค้าง เพื่อเข้าสู่หน้าจอ switch:
  3. พิมพ์ SWITCH_IGNORE_STARTUP_CFG=1
  4. พิมพ์ boot
  5. พิมพ์ enable เมื่อบูตเสร็จ
  6. รัน copy startup-config running-config
  7. เข้าโหมดคอนฟิกและตั้งรหัสผ่านใหม่
  8. พิมพ์ no system ignore startupconfig switch all
  9. บันทึกด้วย copy running-config startup-config

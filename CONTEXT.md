# Context: Cisco Learn Portal & Network Visualizer

ความเข้าใจร่วมกันในคำศัพท์เชิงโดเมน (Glossary) และสถาปัตยกรรมสำหรับระบบเรียนรู้และระบบจำลองเครือข่าย

## 📖 Glossary (ศัพท์บัญญัติโดเมน)

### 1. CLI Terminal State (สถานะเทอร์มินัล CLI)
* **นิยาม**: ออบเจกต์ในหน่วยความจำ (JavaScript State Object) ที่เก็บข้อมูลการตั้งค่าปัจจุบันของ Cisco Switch เช่น ชื่อเครื่อง (Hostname), สถานะของพอร์ตแต่ละช่อง (Up/Down), การแมปพอร์ตกับ VLAN (Access/Trunk), กุญแจความปลอดภัย (RSA Keys), และรหัสผ่านที่ตั้งค่าไว้
* **หน้าที่**: เป็นแหล่งข้อมูลความจริงหนึ่งเดียว (Single Source of Truth) สำหรับการจำลองระบบปฏิบัติการ Cisco IOS

### 2. Network Diagram Component (แผนภาพจำลองเครือข่าย)
* **นิยาม**: ส่วนแสดงผลหน้าเว็บ (มักจะวาดด้วย SVG หรือ HTML5) ที่แสดงผลภาพของอุปกรณ์เครือข่าย ได้แก่ สวิตช์กลาง (Switch), เครื่องคอมพิวเตอร์ PC, พอร์ตเชื่อมต่อ และสายเชื่อมโยงสัญญาณ (Links)
* **หน้าที่**: แสดงผลสถานะเครือข่ายทางกายภาพและโลจิกให้ผู้เรียนเห็นภาพการเปลี่ยนแปลงทันที

### 3. State Binder (ตัวประสานสถานะ)
* **นิยาม**: ตัวตรวจจับ (Listener) หรือตัวกลางในการสื่อสารที่คอยฟังการเปลี่ยนแปลงของ **CLI Terminal State** แล้วสั่งอัปเดตการวาดผลของ **Network Diagram Component** โดยอัตโนมัติ
* **หน้าที่**: ช่วยแยกส่วนแสดงผล (UI) ออกจากส่วนประมวลผลคำสั่ง (Parser Logic) เพื่อให้โค้ดมีความเป็นโมดูลและแก้ไขได้ง่าย

### 4. Port Link State (สถานะลิงก์ของพอร์ต)
* **นิยาม**: สถานะทางภาพของสายเชื่อมโยงระหว่างพอร์ตแลนของ PC และสวิตช์ ซึ่งเปลี่ยนสถานะตามคำสั่ง CLI:
  * **ทางกายภาพ (Link Cable & Port LED)**:
    * **Up (เปิดใช้งาน)**: สายเคเบิลเป็นสีเขียว (หรือสีตาม VLAN) และไฟ LED บนพอร์ตสวิตช์เป็นสีเขียว ➡️ บ่งชี้ว่าอุปกรณ์พร้อมสื่อสาร
    * **Down / Shutdown (ปิดใช้งาน)**: สายเคเบิลเป็นสีแดง และไฟ LED บนพอร์ตสวิตช์เป็นสีแดง ➡️ บ่งชี้สถานะ Administratively Down
  * **ทางตรรกะ (VLAN Membership & ID Badge)**:
    * เมื่อพอร์ตสังกัดอยู่ใน VLAN ที่ไม่ใช่ Default (VLAN 1):
      * สายเคเบิลจะเปลี่ยนสีไปตามรหัสสีของ VLAN นั้น ๆ (เช่น VLAN 10 = สีม่วง, VLAN 20 = สีส้ม)
      * แสดงป้ายกำกับ (VLAN ID Badge) เช่น `[V10]` ปรากฏเด่นชัดข้างพอร์ตเชื่อมต่อบนแผนภาพ

# บันทึกการสอน (Teacher Notes)

## ข้อมูลและความชอบของผู้เรียน (User Preferences)
- **ภาษาที่ใช้สอน:** ภาษาไทย (Thai language) เป็นหลัก เพื่อให้เข้าใจง่ายและครอบคลุม
- **เป้าหมายหลัก:** ต้องการเรียนรู้การ Config Cisco Network Switch และ Network ตั้งแต่ระดับพื้นฐาน (Basic) ไปจนถึงคำสั่งที่ใช้บ่อย และการทำ Password Recovery
- **สิ่งที่ขอมาในเซสชันแรก:**
  1. ลิสต์หัวข้อที่ควรเรียนรู้โดยเรียงลำดับจากระดับ Basic ขึ้นไป
  2. ชุดคำสั่งพื้นฐาน (Basic Commands) ที่ใช้บ่อยพร้อมคำอธิบาย
  3. ขั้นตอนการ Reset / Recover Password ตามมาตรฐานของ Cisco
  4. ลิงก์อ้างอิงจากเว็บอย่างเป็นทางการและน่าเชื่อถือ

## แผนการสอน (Curriculum Outline)
- **Lesson 0001 (บทเรียนแรก):** ปฐมนิเทศ และภาพรวมคำสั่งพื้นฐาน + วิธีการกู้คืนรหัสผ่าน (สิ่งที่จะทำในวันนี้)
- **Lesson 0002:** การทำงานของ Cisco CLI และโหมดต่างๆ (User EXEC, Privileged EXEC, Global Config)
- **Lesson 0003:** การตั้งค่าพื้นฐานสำหรับความปลอดภัย (Enable Secret, Local User, Line Console, Line VTY)
- **Lesson 0004:** การเชื่อมต่อจากระยะไกลและการตั้งค่า IP Management (VLAN 1, Default Gateway, SSH 2.0)
- **Lesson 0005:** เรื่องของ VLAN และ Trunking (802.1Q)
- **Lesson 0006:** การกู้คืนรหัสผ่านสวิตช์แต่ละรุ่นในทางปฏิบัติ (ROMMON / Mode Button)
- **Lesson 0007:** การสำรองข้อมูลและการกู้คืน Configuration (TFTP, FTP, Copy Start Run)

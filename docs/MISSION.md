# Mission: การตั้งค่าสวิตช์และเครือข่าย Cisco (Cisco Switch & Network Configuration)

## Why (ทำไมถึงเรียน)
เพื่อสร้างความรู้และทักษะพื้นฐานที่มั่นคงในการตั้งค่า (Configuration) และบริหารจัดการอุปกรณ์เครือข่ายของ Cisco (โดยเฉพาะ Switch) เพื่อสามารถออกแบบ ติดตั้ง ควบคุมความปลอดภัย และแก้ไขปัญหาในระบบเครือข่ายระดับองค์กรได้อย่างถูกต้องตามมาตรฐาน

## Success looks like (เป้าหมายความสำเร็จ)
- เข้าใจโครงสร้าง Command Line Interface (CLI) ของ Cisco IOS และสลับโหมดการทำงานได้อย่างคล่องแคล่ว
- สามารถตั้งค่าพื้นฐานของ Switch เช่น Hostname, Passwords, Management IP, Default Gateway ได้อย่างถูกต้อง
- สามารถเปิดใช้งาน SSH เพื่อการเข้าถึงจากระยะไกลอย่างปลอดภัย (Secure Remote Access)
- สามารถเข้าใจและกู้คืนรหัสผ่าน (Password Recovery/Reset) ของสวิตช์ Catalyst ตามมาตรฐานของ Cisco เมื่อเกิดเหตุฉุกเฉินได้
- สามารถตั้งค่า VLAN และ Trunking เบื้องต้นเพื่อแบ่งแยกเครือข่ายได้
- รู้วิธีตรวจสอบสถานะ (Verification) และบันทึกค่าคอนฟิกูเรชัน (Save Configuration)

## Constraints (ข้อจำกัด / รูปแบบการเรียนรู้)
- เรียนรู้ผ่านเอกสาร บทเรียน (Lessons), ชีทสรุปคำสั่ง (Reference sheets) และแบบทดสอบ (Quizzes) ในรูปแบบภาษาไทย
- อ้างอิงเนื้อหาตามมาตรฐานสากลและเอกสารอย่างเป็นทางการของ Cisco

## Out of scope (สิ่งที่ยังไม่เน้นในตอนนี้)
- การตั้งค่าเราเตอร์ขั้นสูง (Advanced Routing Protocols เช่น OSPF, BGP)
- เทคโนโลยี Enterprise ขนาดใหญ่ (SD-WAN, Cisco DNA Center)
- การตั้งค่าความปลอดภัยขั้นสูง (เช่น 802.1X, Dynamic ARP Inspection) เพื่อโฟกัสที่พื้นฐานสำคัญก่อน
